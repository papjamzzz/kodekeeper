Kode Keeper — User Guide
USAGE LEFT
Session Context Remaining — how much room is left in this Claude window. At 70% used, start wrapping up. At 85%+ used, Claude's recall degrades — start a fresh session. Month Tokens — total volume consumed this month.
CONTEXT BAR
Fine-grained real-time view of the session fill. Watch the ASCII bar grow. The cursor blinks green until you need to worry.
SYSTEM
CPU — spike here means something is hammering your machine. RAM — above 80% = slowdowns. Swap — overflow from RAM; any swap usage is a warning sign. Disk — low disk causes silent failures and crashed git pushes. Battery — % and minutes to full (0 = charged).
TOK/MIN
Token velocity — how fast you're consuming context in this session. High = deep work. Suddenly drops to zero = you're idle or Claude is waiting on you.
PROJECTS
Green dot = server running. Click OPEN → to go there instantly. Offline = run make run in that folder.
GIT
✓ clean = committed & pushed. ● dirty = uncommitted work on disk — go push it before you lose it.
