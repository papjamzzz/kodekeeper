Kode Keeper — User Guide
USAGE LEFT
Two meters: Session Context (how full this Claude window is) and Month Tokens (total volume this month). Session context is the one to watch — when it hits 70%+ start wrapping up work.
CONTEXT BAR
Real-time fill of the current session window. 70% = start wrapping up. 85%+ = Claude's memory degrades — start a new session immediately.
SYSTEM
RAM — how much memory is in use. Above 80% = things start slowing down. Above 90% = crashes possible. Battery — current % and minutes to full charge (0 = fully charged).
PROJECTS
Every app in your stack. Green dot = running now. Click OPEN → to launch it. Offline = run make run in that folder.
GIT
✓ clean = committed & pushed. ● dirty = uncommitted changes sitting on your machine. Go push before you lose it.
Dashboard refreshes every 6 seconds. AUTO-OPEN LED (amber) toggles browser auto-launch on startup. Built by Creative Konsoles.
